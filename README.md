# House_Price_Prediction_System 🏠

This project is a **Regression-based Machine Learning** application developed during my internship. It predicts the median house value in California districts based on economic and geographic data.

## 🚀 Project Overview
The goal of this model is to estimate real estate prices by analyzing features like median income, house age, and location. This project demonstrates the transition from basic linear models to more complex **Ensemble Learning** techniques.

## 🛠️ Tech Stack
*   **Language:** Python
*   **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn
*   **Algorithm:** Random Forest Regressor

## 📊 Key Features & Insights
*   **Data Source:** Scikit-learn's California Housing dataset.
*   **Feature Importance:** Identified **Median Income** and **Location (Lat/Long)** as the primary drivers of housing prices.
*   **Evaluation:** The model's performance was measured using **R² Score** and **Mean Absolute Error (MAE)**.

## 📈 Visualizations
*   **Correlation Heatmap:** To understand the relationship between different geographic factors.
*   **Feature Importance Bar Chart:** Visual proof of which variables actually influence the house prices.
